Mis proyectos recientes

BURGER'S HOUSE

He participado activamente en el diseño y desarrollo de un sitio e-commerce, aplicando un profundo conocimiento en tecnologías web como HTML, CSS, Bootstrap 5, JavaScript y Java, así como en bases de datos con MySQL. En este proyecto, he utilizado herramientas de desarrollo y gestión de proyectos como Postman para el manejo de APIs, además de Git y GitHub para el control de versiones y colaboración en equipo. Mi enfoque ha sido siempre en la creación de interfaces intuitivas y en la implementación de soluciones eficientes que mejoren la experiencia del usuario y optimicen los procesos de negocio.

Home Blooms

Para la página de plantas, utilicé HTML y CSS para crear un sitio web visualmente atractivo y fácil de usar. Una página dedicada a las plantas es esencial para cualquier persona apasionada por las plantas, ya que no solo ofrece una plataforma para comprar una variedad de plantas, sino que también proporciona una sección de blog con información valiosa sobre el cuidado de las plantas, nutrición y control de plagas. Esta combinación asegura que los usuarios tengan todos los recursos que necesitan para cultivar y mantener sus plantas con éxito. Además, usé Git y GitHub para el control de versiones y la colaboración, garantizando que el proceso de desarrollo fuera eficiente y organizado.

Clon de Google

Para crear mi clon de Google, utilicé HTML para estructurar la página, CSS para aplicar estilos que replican el diseño original, y algo de JavaScript para funcionalidades básicas como la interacción con la barra de búsqueda. Este proyecto me permitió perfeccionar mis habilidades en diseño web responsivo y accesible, además de practicar la gestión de versiones y colaboración utilizando Git y GitHub.
